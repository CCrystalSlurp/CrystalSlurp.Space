Himari-Bot Command List Socials

Command List (They are case sensitive)
Command Use
 !Bees  Sends a message of the first few lines of the "Bee Movie" Script
 !8Ball  Just like a normal 8ball ask a question and it will give an answer
 !add  Does basic addition of two provided numbers
 !ban  Bans mentioned user
 !clear  Purges a given amount of messages
 !help  Brings you here
 !hug  Sends a gif of a hug
 !kick  Kicks the mentioned user from the server
 !kiss  Sends a gif of a kiss
 !monke  Sends funny monke gif
 !multiply  Does basic multiplication
 !mykull  Sends a gif of Michael Reeves
 !ping  Check delay/latency
 !remainder  Does some math thing idk
 !repeat  Copies your message and repeats it
 !subtract  Does basic subtraction
 !unban  Unbans user
 Music Commands
 connect  connects Himari to your voice channel
 disconnect  disconnects music bot from current voice channel
 play  play the given song
 skip  skips the song currently playing
 pause  pauses the song where it is
 resume  continues from where the music was paused
